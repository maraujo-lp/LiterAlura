LiterAlura

Desenvolvido para o Challenge da Oracle ONE Alura.

x Sobre o Projeto

LiterAlura é um programa em Java que permite buscar livros por meio da API Gutendex e guarda em um banco de dados PostgreSQL.

x Funcionalidades

    Buscar livros por título (API Gutendex)
    Procura todos os livros salvos
    Procura todos os autores
    Procura autores vivos em determinado ano
    Procura livros por idioma

x Tecnologias Utilizadas

    Java 23
    Spring Boot
    Spring Data JPA
    PostgreSQL
    Maven


x Pré-requisitos

    Java 23
    PostgreSQL 
    Maven 

x Passo a passo

    Clone o repositório
    Crie um banco de dados chamado literalura no PostgreSQL
    Configure o arquivo application.properties com suas credenciais do PostgreSQL
    Execute o projeto
